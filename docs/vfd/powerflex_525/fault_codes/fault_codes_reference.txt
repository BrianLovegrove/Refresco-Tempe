VFD POWERFLEX 525 - FAULT CODE REFERENCE
=========================================

Common Fault Codes:

F001 - Motor Overload
  Cause: Motor current exceeds rated value
  Action: Check motor connections, reduce load

F002 - Overvoltage
  Cause: DC bus voltage too high
  Action: Check input voltage, install brake resistor

F003 - Undervoltage
  Cause: DC bus voltage too low
  Action: Check input voltage and connections

F007 - Motor Phase Loss
  Cause: One or more motor phases disconnected
  Action: Check motor wiring and connections

F013 - Motor Overtemperature
  Cause: Motor temperature sensor indicates overheating
  Action: Check cooling, reduce load, check filters

F014 - Drive Overtemperature
  Cause: Drive heatsink temperature too high
  Action: Check ambient temperature, clean filters

F068 - Parameter Checksum
  Cause: Parameter memory corruption
  Action: Reset to factory defaults and reprogram

For complete fault code list, refer to manual section 4.2
